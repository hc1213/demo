面向对象编程

js的精髓或者说重点就是原型集成了。不同于java的class，js没有类的概念。定义类是通过原型继承来实现的。


比如说要创建一个叫 dshu 的学生。但是没有student 的class，这个要怎么创建呢。

        
            我们可以这样:
            var Student = {
                name:'studentName',
                gender:'studentMale',
                age:'studentAge',
                say:function(){
                    console.log('我的名字是:',this.name)
                }
            }//定义一个Student

这时候可以来创建一个dshu的对象

        
            var dshu = {
                name:'dshu'
            };
            dshu.__proto__ = Student;

JavaScript 对每个创建的对象都会设置一个原型，指向它的原型对象。

        
            var arr = [1,2,3];
            原型链是  arr---> Array.prototype ---> Object.prototype ---> null

            var foo = v =>v
            原型链是 foo ---> Function.prototype ---> Object.prototype ---> null

下面开始，我们来用一种构造函数的思想来创建对象

        
            function Student(name){
                this.name = name;
                this.say = function(){
                    console.log('hello,',this.name)
                }
            }

到目前为止，这只是一个简单的函数，但是我们可以用 new关键字来调用这个函数，借此创建一个新的对象

        
            var dshu = new Student('dshu');
            此时dshu 的原型链是
            dshu ---> Student.prototype ---> Object.prototype ---> null
            dshu.constructor == Student;//return true
